Bootstrap – Cards (tarjetas) · Web autoexplicativa

1. ¿Qué es una card en Bootstrap?

Las cards (tarjetas) son bloques de contenido con borde, sombra opcional, imagen, título, texto y botones. Son la base de muchos diseños modernos: tiendas, blogs, portfolios, etc.

Aquí veremos: .card, .card-body, .card-img-top, .card-title, .card-text y cómo crear grids responsivos de cards.

2. Estructura básica de una .card

La estructura mínima de una card es: 

<div class="card">
  <div class="card-body">
    <h5 class="card-title">Título de la tarjeta</h5>
    <p class="card-text">
      Texto de ejemplo dentro de la tarjeta.
    </p>
    <a href="#" class="btn btn-primary">Acción</a>
  </div>
</div>

3. Añadir una imagen: .card-img-top

Para colocar una imagen en la parte superior de la tarjeta se utiliza .card-img-top en un elemento <img>. 

<div class="card">
  <img src="imagen.jpg" class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Producto X</h5>
    <p class="card-text">Descripción breve del producto.</p>
    <a href="#" class="btn btn-sm btn-primary">Ver más</a>
  </div>
</div>
Producto de ejemplo

Esta tarjeta simula un producto de tienda online: imagen, título, descripción breve y un botón de acción.

4. .card-title y .card-text

Dentro de .card-body se usan:

  • .card-title → para el título principal.
  • .card-text → para el contenido de texto normal.
<div class="card-body">
  <h5 class="card-title">Artículo del blog</h5>
  <p class="card-text">
    Un pequeño resumen del contenido del artículo.
  </p>
  <a href="#" class="btn btn-outline-primary btn-sm">Leer más</a>
</div>

También se suelen combinar con utilidades de Bootstrap, como .text-muted para fechas o categorías: 

<p class="card-text">
  <small class="text-muted">Publicado el 26/11/2025</small>
</p>

5. Cards + Grid: diseños tipo tienda / blog

La verdadera potencia aparece cuando combinamos las tarjetas con el sistema de grid: .row + .col-*. Así creamos listados responsivos de productos, posts, etc.

Ejemplo de tres cards en una fila responsiva:

<div class="row g-4">
  <div class="col-12 col-md-6 col-lg-4">
    <div class="card">...card aquí...</div>
  </div>

  <div class="col-12 col-md-6 col-lg-4">
    <div class="card">...card aquí...</div>
  </div>

  <div class="col-12 col-md-6 col-lg-4">
    <div class="card">...card aquí...</div>
  </div>
</div>
  • col-12 → en móvil, cada card ocupa todo el ancho.
  • col-md-6 → a partir de pantallas medianas, 2 cards por fila.
  • col-lg-4 → en pantallas grandes, 3 cards por fila.
  • g-4 → separación (gap) entre cards.
